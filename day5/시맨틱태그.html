시맨틱 태그

시멘틱 태그는 HTML 요소의 의미를 명확하게 나타내는 태그
시멘틱 태그를 사용하면 문서 구조를 이해하기 쉽고 검색 엔진이 페이지의 콘텐츠를 더 잘 이해할 수 있음 또한 웹 접근성을 향상시키고 스타일링 및 레이아웃 작업을 보다 쉽게 수행할 수 있다.

시맨틱 태그 종류

header :
머리말을 나타냄, 보통 웹페이지 상단에 배치되며 사이트 로고, 제목, 검색폼 등을 포함할 수 있다
nav :
문서에서 네비게이션 링크 영역을 나타낸다, 주메뉴와 링크그룹을 포함할 수 있다
main :
문서의 주요 콘텐츠 포함하는 영역을 나타낸다, 한페이지에 메인태그는 하나만 사용한다
주로 본문 내용을 감싸는 역할을 한다
article :
독립적인 콘텐츠 조각을 나타낸다, 뉴스 기사, 블로그 글, 게시물등과 같은 독립적인 콘텐츠 영역에 사용됨
section :
서로 관련된 콘텐츠들을 그룹화하는데 사용함
aside :
현재 웹사이트에서 제공하는 내용과는 다른 내용을 제공할 때 사용
예를들면 뭐 광고라던가 사이드바, 광고영역, 다른 부가정보
footer :
문서나 섹션의 바닥글을 나타냄(연락처 정보, 저작권 정보 등을 포함할 수 있다)
